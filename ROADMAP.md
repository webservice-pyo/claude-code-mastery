# 개발자 웹 이력서 프로젝트 로드맵

## 프로젝트 개요
HTML, CSS, JavaScript, TailwindCSS를 이용한 개발자용 웹 이력서 포트폴리오 사이트 구축

---

## Phase 1: 프로젝트 설정 및 기초 구조 (1-2일)

### 1.1 프로젝트 초기화
- [ ] 프로젝트 폴더 구조 생성
  ```
  project/
  ├── index.html
  ├── styles/
  │   └── style.css
  ├── scripts/
  │   └── main.js
  ├── assets/
  │   ├── images/
  │   └── icons/
  └── README.md
  ```
- [ ] Git 초기화 및 .gitignore 설정
- [ ] TailwindCSS 설정 (CDN 또는 npm)

### 1.2 기본 HTML 구조
- [ ] HTML5 기본 템플릿 작성
- [ ] 메타 태그 설정 (SEO, viewport)
- [ ] 폰트 설정 (Google Fonts)
- [ ] TailwindCSS 링크 추가
